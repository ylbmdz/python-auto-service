# Python 自动化脚本服务 🚀
承接各类 Python 轻量脚本开发，自动报价、自动交付！

## 服务范围
- ✅ **Python 脚本**（简单需求，<300 Token）：¥2 起
- ✅ **文案/翻译**（<2000 Token）：¥8 起
- 支持：自动化办公、数据处理、小工具开发等

## 下单方式
1. 点击上方 **「Issues」** → 「New issue」
2. 标题格式：`[订单] 类型:Python脚本,报价:¥X`
3. 内容里写清楚你的需求
4. 系统会自动估算 Token 并校验报价，合规即接单交付！

## 示例
**标题**：`[订单] 类型:Python脚本,报价:¥2`
**内容**：`写一个简单计算器脚本，支持加减乘除运算`

## 注意事项
- 仅接受明确、可自动化的小需求
- 复杂需求（>3 行描述）会被拒单
- 交付后可在 Issue 下查看结果
